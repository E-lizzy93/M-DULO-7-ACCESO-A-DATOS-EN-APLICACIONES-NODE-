🔗 Asignar actor a película

ℹ️ ¿Cómo funciona la transacción?

Al asignar un actor a una película, el servidor usa una transacción:


1. BEGIN — inicia la transacción

2. Inserta el registro en peliculas_actores

3. COMMIT — confirma si todo salió bien

4. ROLLBACK — revierte si hubo error


Si el actor ya está asignado a esa película, el servidor responde con error 409.
